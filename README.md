# Integration of Facebook SDK and Event Tracking in Flutter

This repository contains the specifications and guidelines for integrating the Facebook SDK into our Flutter mobile application. The goal is to enable Meta Ads (formerly Facebook Ads) for our app by setting up comprehensive event tracking related to food delivery and table reservation functionalities.
